# Customer Segmentation Using K-Means Clustering

## Overview
This project analyzes consumer purchasing behavior to segment bath soap customers into meaningful groups for targeted marketing.

## Tools Used
- R
- K-means Clustering
- Data Preprocessing
- Decision Tree Model

## Dataset
- 600 households
- 46 variables including demographics and purchase behavior

## Key Features Engineered
- Brand loyalty (inverse brand count)
- Switching rate
- Herfindahl index

## Methodology
- Data cleaning and preparation
- Feature engineering
- Data scaling
- K-means clustering (K = 3 using elbow and silhouette methods)
- Classification model for validation

## Results
- Identified 3 customer segments:
  - Heavy loyal buyers
  - Light switching users
  - Promotion-sensitive buyers
- Model accuracy: 80%

## Business Insights
- Target loyal customers with rewards programs
- Use promotions for price-sensitive segments
- Focus on retention during economic downturns
